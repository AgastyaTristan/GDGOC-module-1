# Summer Music Fest 2026

Website informasi konser single-page untuk Hands-On Module 1 WebDev GDGOC ITB.

## Struktur Halaman

- **Home** — Nama konser dan tagline
- **About** — Deskripsi festival
- **Date** — Tanggal, waktu, dan lokasi
- **Footer** — Kontak (email, Instagram, WhatsApp)

## Fitur

- HTML5 semantik (`header`, `nav`, `main`, `section`, `footer`)
- Layout Flexbox & CSS Grid
- Mobile responsive (breakpoint 768px & 480px)
- CSS Custom Properties (variables)
- Dark / Light mode toggle
- Animasi hero fade-in dan nav underline hover

## Cara Menjalankan

1. Clone atau buka folder proyek ini
2. Buka `index.html` di browser modern (Chrome, Firefox, Edge)

Tidak ada dependency atau build step — cukup HTML + CSS.
